# Домашнее задание
## Условие задачи
## Сигнализация в музее
### В зале музея четыре датчика движения (A, B, C, D). Чтобы избежать ложных срабатываний, сигнализация включает запись только если не менее трех датчиков сработали (система интерпретирует срабатывания как "повторяющееся движение", а не "шум"). Запишите условие включения записи.
## 1. Алгоритм и блок-схема
### Алгоритм
1. **Начало**
2. Ввести значения четырех датчиков:
* A - состояние датчика 1 (1 - движение, 0 - шум)
* B - состояние датчика 2 (1 - движение, 0 - шум)
* C - состояние датчика 3 (1 - движение, 0 - шум)
* D - состояние датчика 4 (1 - движение, 0 - шум)
3. Проверить условие включения записи:
* Сигнализация срабатывает, если не менее трех датчиков показывают "1"
* Это эквивалентно проверке всех возможных комбинаций, где три или четыре датчика сработали
6. Установить значение сигнализации:
* alarm = 1 (сигнализация включена), если условие выполняется
* alarm = 0 (сигнализация выключена), если условие не выполняется
9. Вывести результат:
* Показать состояние записи (1 - активна, 0 - не активна)
10. **Конец**

 ### Реализация программы
 #define _CRT_SECURE_NO_DEPRECATE
#include <locale.h>
#include <stdio.h>
int main()
{
    setlocale (LC_CTYPE, "RUS");
	int A;
	int B;
	int C;
	int D;
	int alarm;
	printf("Введите 4 числа 1 или 0 (срабатывание датчика 1-повторяющееся движение, 0-шум):\n");
	scanf("%d %d %d %d", &A, &B, &C, &D);
	alarm = (A == 1) && (B == 1) && (C == 1) && (D == 0) || (A == 0) && (B == 1) && (C == 1) && (D == 1) || (A == 1) && (B == 0) && (C == 1) && (D == 1) || (A == 1) && (B == 1) && (C == 1) && (D == 0) || (A == 1) && (B == 1) && (C == 1) && (D == 1);
	printf("Запись активна(1-да,0-нет):%d\n", alarm);
}
